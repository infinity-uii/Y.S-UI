RAG Backend
Current: In-memory (default). For production, configure ChromaDB, Qdrant, PGVector, or Milvus via environment variables:
  • CHROMA_HOST / CHROMA_PORT — ChromaDB connection
  • QDRANT_URL / QDRANT_API_KEY — Qdrant connection
  • PGVECTOR_URL — PostgreSQL with pgvector extension
  • MILVUS_HOST / MILVUS_PORT — Milvus connection
Deployment
Supported deployment targets:
  • 🚂 Railway: Set PORT env var, push to GitHub, connect repo in Railway
  • 🐳 Docker: docker build -t agent-system . && docker run -p 8080:8080 agent-system
  • ☸️ Kubernetes: Use provided Helm chart or manifest templates
  • 📦 pip: pip install -r requirements.txt && python agent_system.py
